monitor
=======

esup portlet monitor

projet based on esup-commons V2

Package installation :
mvn clean install

Packages generation :
mvn clean package

Releases management :
mvn release:prepare
mvn release:perform

Prototyping :
 cd portlet-monitor-web-springmvc-portlet ; mvn portlet-prototyping:run ; cd -
 and go on http://localhost:8080/pluto
